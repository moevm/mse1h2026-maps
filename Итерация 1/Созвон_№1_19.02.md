# Созвон с заказчиком №1
## Агенда
- **Дата**: 19.02.2026
- **Тема встречи**: Первичное обсуждение проекта
- **План встречи**:
  - Язык ввода запроса.  **Итог:** не имеет значения.
  - Приёмочные тесты (испытания).  **Итог:** тестирование происходит по каждой функциональности: 
    - поиск по запросу пользователя;
    - визуализация найденной информации в виде;
    - взаимодействие с графом.
  - Представление информации, полученной при взаимодействии пользователя с графом.  **Итог:** дополнительная всплывающая панель, на которой содержится основная информация об узле графа: 
    - описание сущности;
    - основные связи;
    - несколько ссылок на источники информации (статьи).
  - Возможность выбора источников информации.  **Итог:** добавить пользователю возможность выбора источников информации. При этом информацию из невыбранных источников обрабатывать, но пользователю не показывать (не выводить на граф). 
  - Ограничение количества узлов.  **Итог:** создать возможность скрывать выбранные типы связей. 
  - Нефункциональные требования.  **Итог:** должны выполняться требования ко всем слоям прозрачности.
